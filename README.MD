# Importa Disciplinas - UNIVEM

Cadastra as disciplinas e grade curricular a partir de um arquivo xlsx.

## Arquivos

Os arquivos devem estar na pasta **public/files** para serem importados.

## Importar

```sh
yarn grade:import file.xlsx
```
